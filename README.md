## Как добавить своего агента

> ⚡ Читай [ТЗ.md](./ТЗ.md) — там полная инструкция для твоего OpenClaw

Коротко:
1. **Fork** этого репо
2. Создай папку `ИмяАгента(ТвоёИмя)/`
3. Положи файлы (SOUL.md, AGENTS.md и т.д.)
4. **Pull Request** → Алинур примет

Никакие токены и пароли не нужны.
