# 📋 Инструктаж: Как поделиться опытом агента

## Что это такое

Публичный репозиторий где каждый делится опытом своего OpenClaw агента.
Твой агент может научить другого агента чему-то полезному.

## Шаг 1 — Что можно экспортировать

```
✅ МОЖНО (опыт и знания):
  SOUL.md           → характер агента
  AGENTS.md         → правила поведения
  HEARTBEAT.md      → что проверяет в фоне
  agents/*/profile.md → профили суб-агентов

❌ НЕЛЬЗЯ (личное и секреты):
  TOOLS.md          → там токены и пароли
  USER.md           → личные данные
  .env файлы        → секреты
  личная переписка  → приватно
```

## Шаг 2 — Создай папку с именем агента

Формат: `ИмяАгента(ТвоёИмя)`

Примеры:
- `Hamori(Alinur)`
- `Atlas(Maria)`
- `Rex(Denis)`

## Шаг 3 — Скопируй файлы в папку

```bash
mkdir "ИмяАгента(ТвоёИмя)"
cp ~/openclaw/SOUL.md "ИмяАгента(ТвоёИмя)/"
cp ~/openclaw/AGENTS.md "ИмяАгента(ТвоёИмя)/"
cp ~/openclaw/HEARTBEAT.md "ИмяАгента(ТвоёИмя)/"
cp -r ~/openclaw/agents/ "ИмяАгента(ТвоёИмя)/agents/"
```

## Шаг 4 — Проверь перед отправкой

Открой каждый файл и убедись что:
- Нет API токенов
- Нет паролей
- Нет личных телефонов/адресов

## Шаг 5 — Отправь

GitHub

```bash
git clone https://github.com/Alinur941/openclaw-agents-hub.git
cd openclaw-agents-hub
cp -r "ИмяАгента(ТвоёИмя)" .
git add .
git commit -m "Добавлен агент ИмяАгента(ТвоёИмя)"
git push
```
