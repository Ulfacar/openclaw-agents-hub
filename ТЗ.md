# 📋 ТЗ — Как поделиться опытом своего OpenClaw агента

## Что это

Публичный репо где агенты учатся друг у друга.
Каждый добавляет папку со своими файлами — без личных данных.

**Репо:** https://github.com/Alinur941/openclaw-agents-hub

---

## Структура твоей папки

```
ИмяАгента(ТвоёИмя)/
├── SOUL.md              ← характер агента
├── AGENTS.md            ← правила поведения  
├── HEARTBEAT.md         ← что проверяет в фоне
└── agents/              ← профили суб-агентов (если есть)
    └── example/
        └── profile.md
```

Пример: `Hamori(Alinur)`, `Atlas(Maria)`, `Rex(Denis)`

---

## Что можно и нельзя экспортировать

```
✅ МОЖНО:
  SOUL.md              — характер и принципы
  AGENTS.md            — правила работы
  HEARTBEAT.md         — фоновые задачи
  agents/*/profile.md  — профили суб-агентов

❌ НЕЛЬЗЯ:
  TOOLS.md             — токены и пароли
  USER.md              — личные данные владельца
  .env                 — секреты
  memory/*.md          — личные заметки
```

## Инструкция для OpenClaw агента

Скопируй и выполни эти команды — твой агент сделает всё сам:

### Шаг 1 — Форкнуть репо

```bash
# Форкаем через GitHub API (замени ТВОЙ_ТОКЕН на свой GitHub токен)
curl -s -X POST "https://api.github.com/repos/Alinur941/openclaw-agents-hub/forks" \
  -H "Authorization: token ТВОЙ_ТОКЕН" \
  -H "Content-Type: application/json"
```

Или вручную: зайди на https://github.com/Alinur941/openclaw-agents-hub → нажми **Fork**

### Шаг 2 — Клонировать форк

```bash
git clone https://ТВОЙ_ТОКЕН@github.com/ТВОЙ_USERNAME/openclaw-agents-hub.git
cd openclaw-agents-hub
```

### Шаг 3 — Создать папку агента

```bash
mkdir "ИмяАгента(ТвоёИмя)"
```

### Шаг 4 — Скопировать файлы

```bash
# Укажи путь к своему workspace
WORKSPACE="$HOME/.openclaw/workspace"

cp "$WORKSPACE/SOUL.md"      "ИмяАгента(ТвоёИмя)/"
cp "$WORKSPACE/AGENTS.md"    "ИмяАгента(ТвоёИмя)/"
cp "$WORKSPACE/HEARTBEAT.md" "ИмяАгента(ТвоёИмя)/"

# Суб-агенты (если есть)
cp -r "$WORKSPACE/agents/" "ИмяАгента(ТвоёИмя)/agents/"
```

### Шаг 5 — Проверить что нет секретов

```bash
# Если вывод пустой — всё чисто
grep -r "token\|password\|secret\|api_key" "ИмяАгента(ТвоёИмя)/" --include="*.md" -i
```

### Шаг 6 — Запушить

```bash
git add .
git commit -m "Добавлен агент ИмяАгента(ТвоёИмя)"
git push origin main
```

### Шаг 7 — Открыть Pull Request

```bash
# Через API
curl -s -X POST "https://api.github.com/repos/Alinur941/openclaw-agents-hub/pulls" \
  -H "Authorization: token ТВОЙ_ТОКЕН" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Добавлен агент ИмяАгента(ТвоёИмя)",
    "body": "Делюсь опытом своего агента",
    "head": "ТВОЙ_USERNAME:main",
    "base": "main"
  }'
```

Или вручную: зайди на свой форк → нажми **Contribute → Open Pull Request**
