# 📋 ТЗ — Экспорт опыта OpenClaw агента

## Что это и зачем

Мы собираем знания наших OpenClaw агентов в одном месте.
Цель: учиться друг у друга — как настроен агент, какие суб-агенты есть, как работает.

**Репозиторий:** https://github.com/Alinur941/openclaw-agents-hub  
**Формат участия:** каждый создаёт папку `ИмяАгента(ТвоёИмя)` и кладёт туда свои файлы.

---

## Пример папки

```
Hamori(Alinur)/          ← ИмяАгента(ИмяВладельца)
├── SOUL.md              ← характер агента
├── AGENTS.md            ← правила поведения
├── HEARTBEAT.md         ← что проверяет в фоне
└── agents/              ← профили суб-агентов
    ├── backend/
    │   └── profile.md
    └── frontend/
        └── profile.md
```

---

## Что экспортировать

```
✅ МОЖНО — опыт и знания:
  SOUL.md              → характер и принципы агента
  AGENTS.md            → как агент работает
  HEARTBEAT.md         → что проверяет по расписанию
  agents/*/profile.md  → профили суб-агентов

❌ НЕЛЬЗЯ — личное и секреты:
  TOOLS.md             → там API токены и пароли
  USER.md              → твои личные данные
  .env файлы           → секреты
  memory/*.md          → личные заметки
  любые пароли и токены
```

## Как добавить — 3 способа

### Способ 1 — Через GitHub (если есть аккаунт)

1. Зайди на https://github.com/Alinur941/openclaw-agents-hub
2. Нажми **Fork** (вверху справа)
3. Создай папку `ИмяТвоегоАгента(ТвоёИмя)` в своём форке
4. Добавь файлы
5. Нажми **Contribute → Open Pull Request**
6. Алинур примет изменения

### Способ 2 — Через токен (без Fork)

Используй этот токен для прямого пуша:
```
Токен: [Алинур пришлёт в личку]
```

```bash
git clone https://ТОКЕН@github.com/Alinur941/openclaw-agents-hub.git
cd openclaw-agents-hub
mkdir "ИмяАгента(ТвоёИмя)"
# скопируй файлы в папку
git add .
git commit -m "Добавлен агент ИмяАгента(ТвоёИмя)"
git push
```

### Способ 3 — Отправить Алинуру

Просто заархивируй папку и отправь Алинуру в Telegram.
Он сам добавит в репо.

## Команды для экспорта из OpenClaw

Выполни в терминале где установлен OpenClaw:

```bash
# Создай папку с именем агента
mkdir "ИмяАгента(ТвоёИмя)"

# Скопируй разрешённые файлы
cp ~/.openclaw/workspace/SOUL.md "ИмяАгента(ТвоёИмя)/"
cp ~/.openclaw/workspace/AGENTS.md "ИмяАгента(ТвоёИмя)/"
cp ~/.openclaw/workspace/HEARTBEAT.md "ИмяАгента(ТвоёИмя)/"
cp -r ~/.openclaw/workspace/agents/ "ИмяАгента(ТвоёИмя)/agents/"

# Проверь что нет токенов и паролей!
grep -r "token\|password\|secret\|api_key" "ИмяАгента(ТвоёИмя)/" --include="*.md"
```
